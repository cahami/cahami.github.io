Getting started

This is the remote client for Hadouken, a cross-platform, headless, BitTorrent client. To get started you will need a running Hadouken instance and be able to connect to it from the computer you are using now. You will also need the username/password (if you are using Basic Auth) or token (if you are using Token Auth).
